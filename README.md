# Jvedio-AI
一个本地化的（JAV）电影管理/数据库系统

Jvedio AI是基于原Jvedio开源（C#/.NET）项目（https://github.com/hitchao/Jvedio）修改而来且持续维护的新一代JAV/电影管理系统。

本软件除大量重构和优化了原版软件的代码及功能外，还额外新增了众多实用功能：

1. 基于本地AI模型的影片数据语义分析功能。
2. 基于本地AI模型的使用者用户画像功能。
3. 基于本地AI模型的个性化推荐功能。
4. 基于本地AI模型的同类影片推荐功能。
5. 基于本地AI模型的语义搜索功能。
6. 基于Serverless架构的云端字幕匹配和下载功能。

本项目旨在打造一款主要功能纯本地化、注重隐私、符合当下用户使用场景需求和习惯的影片管理系统。

除继承了Jvedio的整体框架、界面设计外，本项目通过对原项目的代码重构，对内存占用、磁盘I/O线程、UI响应等方面进行了深度优化，实现了功能和性能开销上的平衡。

项目本身并不提供任何资源或资源获取渠道。所有数据均来源于用户本地存储资源，以及互联网公开数据源。
